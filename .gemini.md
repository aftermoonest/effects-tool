# WebGL Image Editor - AI Context

## Project Architecture
- **Framework:** React + Vite + TypeScript
- **UI:** Shadcn/ui & Tailwind CSS
- **State:** Zustand (store state OUTSIDE of React)
- **Engine:** Vanilla TS + `regl` WebGL wrapper (NO heavy 3D frameworks).

## Skills & Principles
You MUST reference the skills in `.agents/skills/` before complex implementations:
- `shader-programming-glsl`: For writing pure Photoshop math GLSL fragment and vertex shaders.
- `zustand-store-ts`: For architecting the `editorStore`.
- `architecture` & `plan-writing`: For structuring large features.
- `frontend-design`: For ensuring Shadcn components look premium and not generic.

## Folder Structure
- `src/engine/`: Vanilla TS classes (`Compositor`, `AsciiRenderer`, shader GLSL strings/files).
- `src/store/`: Zustand definitions.
- `src/components/`: React UI pieces.
- `src/components/ui/`: Auto-generated Shadcn components.
